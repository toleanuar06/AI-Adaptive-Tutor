# AI-Powered Ergonomics & Adaptive Tutor 🤖📚

Бұл жоба — Компьютерлік көру (Computer Vision), C++ логикалық өзегі және динамикалық компиляция арқылы жұмыс істейтін гибридті білім беру және қауіпсіздік экожүйесі.

## 🚀 Жобаның негізгі мүмкіндіктері (Features)

1. **AI Safety & Ergonomics Monitor (Python + C++)**
   * **Object Detection (YOLOv8):** Жұмыс үстеліндегі қауіпті сұйықтықтарды (бөтелке, стақан) және құрылғыларды нақты уақытта таниды. Егер сұйықтық ноутбукке тым жақын тұрса, C++ жүйесі дабыл қағып, ұпай шегереді.
   * **Pose Estimation (YOLOv8-Pose):** Оқушының дене қалпын бақылайды. Экранға тым жақын еңкейіп (slouching) отырса, математикалық есептеулер арқылы анықтап, дыбыстық ескерту жасайды.
   
2. **Adaptive Tutor System (Dynamic Evaluation)**
   * Оқушының білім деңгейіне қарай бейімделетін (Scaffolding) ақылды жүйе.
   * **Sandboxing:** Оқушының жазған кодын Python арқылы жасырын түрде C++ (`g++`) компиляторына жіберіп, синтаксистік және логикалық қателерді нақты компилятор деңгейінде тексереді.

## 🛠️ Қолданылған технологиялар (Tech Stack)
* **Python 3.x:** OpenCV, Ultralytics (YOLOv8), Subprocess (Dynamic execution).
* **C++ (g++):** Математикалық есептеулер, қауіпсіздік логикасы, файлдық жүйемен жұмыс (Дерекқор).
* **AI Models:** `yolov8n.pt`, `yolov8n-pose.pt`.

## ⚙️ Архитектура
Жүйе микросервистік логикамен жұмыс істейді: Python (Vision) камерадан оқылған мәліметтерді локальды файлдар арқылы C++ (Logic) жүйесіне жібереді. C++ шешім қабылдап, UI үшін деректерді кері қайтарады.
